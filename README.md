# UniStyle

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/abaker421)

UniStyle is a free Unicode text formatting tool. Convert plain text into bold, italic, script, monospace, strikethrough, double-struck, vaporwave, small caps, and more. The Unicode output copy-pastes directly into Discord, Notion, Twitter/X, Slack, LinkedIn, email, and anywhere plain text input is accepted.

## Two ways to use it

- **Web app and PWA**: https://unistyle.io - installable on desktop and Android.
- **Chrome extension**: popup and right-click context menu. Coming soon to the Chrome Web Store.

## Why Unicode, not Markdown?

Markdown bold (`**like this**`) only works inside the message body of apps that parse it. UniStyle generates actual Unicode characters, so the formatting travels with the text. It renders in Discord bios, Twitter display names, Notion page titles, Slack status, and third-party clients where Markdown is unavailable.

## What it supports

- 20+ Unicode styles (bold, italic, bold italic, script, fraktur, double-struck, monospace, vaporwave, small caps, strikethrough, underline, overline, and more)
- Style combiner - stack a modifier on any base style
- Sentence cleanup, case conversion, special character insert
- No sign-up, no account, no tracking, no telemetry
